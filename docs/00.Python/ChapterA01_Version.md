python2中不加括号，可以兼容3中加括号，但仅限于只有一个元素，否则可能出错。

### 3. 文件读取时指针区别
python2中用r模式打开的文件可以从当前位置回退指针
python3中必须用rb模式才行。
若要考虑两者兼容性，都统一使用rb模式打开文件就好，然后统一使用bytes.decode()解码，或在str字符串前面加b转换为bytes保持一致
